// HOW IT WORKS

1. Import embed.js — it registers your game (window.location.origin + page <title>).
2. After scene, camera, and player position exist: createVibePortal({ scene, camera }). The portal is a THREE.Group — position it, then scene.add(portal).
3. Every frame: portal.update(player.position) (or any Vector3).
4. Proximity: ~6 world units shows the enter prompt; ~2 units navigates. Distances scale with scale.
5. Optional: set a clear <title> in HTML so other games see a good label above your portal.
6. If the URL has ?ref= (player came from another game), the embed shows a Return portal — no extra code.
Options (defaults): label · username · avatar · scale (1) · proximityDist (6) · enterDist (2) · game ('the-vibe-metaverse')
No PR needed — add the import, placement, and portal.update in your loop.
